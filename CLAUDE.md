## Project Overview

Static website for the **Treasure Valley Vintage Computer Club (TVVCC)** built with Hugo and deployed on Netlify. The site uses the `re-terminal` retro-styled theme (git submodule).

## Commands

### Local Development
```bash
hugo server
```

### Production Build
```bash
hugo --gc --minify --baseURL "https://tvvcc.org/"
```

### Create New Post
```bash
hugo new posts/my-post-title.md
```

## Architecture

- **hugo.toml** — Primary site configuration (base URL, theme settings, menus, pagination). A legacy `config.toml` also exists but `hugo.toml` takes precedence.
- **content/** — Markdown source files. Posts go in `content/posts/`. Standalone pages (like `about.md`) go in the content root.
- **themes/re-terminal/** — Theme (git submodule from `mirus-ua/hugo-theme-re-terminal`). Provides all layouts, styles, and JS.
- **archetypes/default.md** — Template used when creating new content with `hugo new`.
- **public/** — Build output (gitignored). Regenerated by Hugo on build and by Netlify on deploy.
- **netlify.toml** — Build configuration and environment versions (Hugo 0.152.2, Dart Sass 1.93.2, Go 1.25.3, Node 22.20.0).

## Theme Configuration

The site uses the `re-terminal` theme with these key settings in `hugo.toml`:
- Theme color: `"orange"` (options: orange, blue, red, green, pink, paper)
- Center theme enabled, full-width disabled
- Pagination: 5 posts per page
- Reading time display enabled

## Content Frontmatter

Posts use TOML frontmatter (`+++`) with fields: `title`, `date`, `draft`, `tags`, `categories`. The archetype template sets `draft = true` by default.
